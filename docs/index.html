About

What is i3T3?

Manual cell counting is labor-intensive and subject to observer variability. Automated tools like ImageJ require significant manual calibration and often struggle with touching or overlapping cells of irregular form. i3T3 was built to address this using a multi-stage visual reasoning approach.

i3T3 was initially developed and tuned during the Santa Monica College Biotechnology Program's Bio 32 Cell Culture Methods & Techniques course. Crystal Violet stained NIH-3T3 fibroblast culture images from other experiments in the course were used. Once validated, i3T3 was applied as the dry lab / computational component of of the Cell Culture Methods & Techniques final project, which investigated the effects of Indole-3-Acetic Acid (IAA) on NIH-3T3 cell growth in vitro.

The broader goal of i3T3 is to serve as a foundation for a biomedical visual-reasoning model evaluation tool, filling gaps in AI evaluation and model failure analysis for image visual reasoning tasks.

How It Works

Pipeline

i3T3 processes each image through stages, from raw pixel data to a labeled count and a per-object feature export.

Step 01
Image to Matrix

Convert RGB image to 8-bit grayscale matrix via scikit-image.

Step 02
Cell Body Mask

Gaussian blur + Otsu threshold isolates cell bodies from background.

Step 03
Nucleus Mask

Percentile threshold on raw matrix captures the darkest nuclear regions independently.

Step 04
Watershed Separation

Peak local max markers seed watershed, splitting touching and overlapping cells.

Step 05
Fibroblast Filter

Morphological rules (area, eccentricity, solidity, nucleus count) classify true cells.

Step 06
Feature Export

40+ shape, intensity, skeleton, and LBP texture features exported per object to CSV.

Validation

Results

i3T3 was validated against manual hand counts and ImageJ automated counts across 36 Crystal Violet stained images from the IAA final project experiment — 4 conditions × 3 time points (24 hr, 48 hr, 72 hr) × 3 replicates. Correlation analysis was performed in R using linear regression.

Method R² vs. Manual Count Notes
i3T3 0.832 Watershed + nucleus-aware filtering
ImageJ 0.01 Automated particle analysis

The difference in correlation suggests that i3T3’s visual reasoning approach has the potential to be developed into a biomedical visual-reasoning model evaluation tool, filling gaps in AI evaluation and model failure analysis for image visual reasoning tasks.

Feature Extraction

40+ Features Per Object

Each segmented cell is exported with a full morphological profile for downstream ML use.

⬡

Shape

Area, perimeter, eccentricity, major/minor axis, aspect ratio, solidity, extent, circularity, compactness, roundness, convexity, elongation index

◑

Intensity

Mean, min, max, and standard deviation from the grayscale pixel matrix

◎

Nucleus

Nucleus count, nucleus area total, nucleus area ratio, body-to-nucleus ratio

∿

Skeleton

Skeleton length and endpoint count via morphological skeletonization

▦

Texture

Normalized Local Binary Pattern (LBP) histogram — 10 bins, uniform, P=8, R=1

⊹

Geometry

Centroid, orientation, convex area, equivalent diameter, Feret diameter, bounding box, perimeter extreme points

Getting Started

Usage

1. Install dependencies

pip install -r requirements.txt

2. Point the script at your image

Update the three filename references in i3T3.py (each marked # CHANGE THIS for each image):

image_matrx = iio.imread("YOUR_IMAGE.jpg")
np.savetxt("YOUR_IMAGE_matrix_uint8.csv", ...)
...
image = iio.imread("YOUR_IMAGE.jpg")
matrix = np.loadtxt("YOUR_IMAGE_matrix_uint8.csv", ...)
...
image_filename = "YOUR_IMAGE.jpg"

3. Run

python i3T3.py

4. Tune key parameters

# Smoothing — increase sigma for noisier images
blurred = gaussian(matrix, sigma=4.0)

# Nucleus sensitivity — lower percentile = stricter
nucleus_threshold = np.percentile(matrix, 41)

# Watershed — increase min_distance to merge nearby peaks
peak_coords = peak_local_max(distance, min_distance=10, threshold_abs=2)

# Fibroblast rule — adjust bounds for your cell line
fibroblast_rule = (
    area >= 1000 and area <= 800000 and
    aspect_ratio >= 0.10 and aspect_ratio <= 8.0 and
    solidity >= 0.20 and
    n_nuclei >= 1 and n_nuclei <= 2
)
